# User Journeys

## 1. Company Onboarding

```
Homepage → Sign Up (company + admin) → Onboarding (add store) → Invite users → Home (rankings)
```

**Actors**: Company Admin  
**Goal**: Start 3-month free trial and configure first store

## 2. Store Inspection (SV)

```
Tab ③ Start Check → Select store → Select check sheet → Answer questions → Attach photos → Submit → Report generated
```

**Actors**: SV, Inspector  
**Goal**: Complete QSC audit with evidence

## 3. Review & Improve

```
Tab ② Reports → Store report → View issues + AI discussion points → Create improvement case
Tab ④ Improvements → Work case → Submit report → SV approves/rejects → Done
```

**Actors**: Store Manager, SV  
**Goal**: Close the loop from finding to verified fix

## 4. Ranking Review (HQ)

```
Tab ① Home → Filter (store/group/improvement rate, period) → Drill into store report
```

**Actors**: Division head, HQ, SV  
**Goal**: Compare performance and prioritize support

## 5. Admin Management

```
Tab ⑤ My Page → Stores / Users / Groups / Check sheets / Billing
```

**Actors**: Company Admin  
**Goal**: Maintain org structure and subscription
